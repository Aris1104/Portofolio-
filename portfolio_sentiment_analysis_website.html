Sentiment Analysis Project - Indonesian National Team Naturalization

About This Project

This project analyzes public sentiment on social media regarding the naturalization of players in the Indonesian national football team. Using Natural Language Processing (NLP) techniques, the project classifies comments into positive, negative, and neutral sentiment.

Project Objectives

  • Collect social media comments related to player naturalization
  • Clean and preprocess text data
  • Build sentiment classification model
  • Visualize sentiment distribution

Technologies Used

Python Pandas Scikit-learn NLP Machine Learning Data Visualization

Methodology

  1. Data Collection from social media comments
  2. Text preprocessing (cleaning, stopword removal, stemming)
  3. TF-IDF feature extraction
  4. Naive Bayes classification model
  5. Evaluation and visualization

Sample Result

Sentiment distribution example:

  • Positive: 55%
  • Negative: 30%
  • Neutral: 15%
